# Code of Conduct

## 1. 目标

我们希望 Remote Subagent Protocol 社区保持专业、开放、可协作。所有参与者都应在讨论与协作中保持尊重。

## 2. 期望行为

- 讨论聚焦技术与事实，不做人身攻击
- 对新贡献者保持耐心，给出可执行反馈
- 在评审中明确问题与建议，不使用羞辱性表达
- 尊重不同背景、经验与观点

## 3. 不可接受行为

- 侮辱、歧视、威胁、骚扰
- 恶意刷屏、挑衅、引战
- 故意泄露他人隐私信息
- 在协作渠道中发布明显不相关或破坏性内容

## 4. 执行与处理

维护者可对违规行为采取以下措施：
- 口头/书面提醒
- 暂时限制互动
- 关闭相关讨论或移除贡献
- 严重情况下永久移除社区参与资格

## 5. 举报方式

请通过 `SECURITY.md` 中的联系方式联系维护者，并在标题注明 `CoC Report`。

我们会尽快处理并尽量保护举报人隐私。
