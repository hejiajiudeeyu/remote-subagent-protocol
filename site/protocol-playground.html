Remote Subagent Protocol Playground

这是协议主库中的 reference playground。它只演示协议步骤、控制面交互、`L0 local transport` 联调语义、ACK、签名验收与错误分支，不承载 marketplace 排序、定价、转化或运营叙事。

对齐规则：POST /v1/users/register 只创建用户主体并默认激活 buyer；seller 能力在 agent 审核导入成功后激活。 目录接口 GET /v1/catalog/subagents 不返回 delivery_address，Buyer 在拿到 token 后通过 POST /v1/requests/{request_id}/delivery-meta 单次申请投递元数据。当前 playground 的数据面以 L0 local transport 语义为主，不依赖 marketplace 页面。
