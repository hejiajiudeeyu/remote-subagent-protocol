# Active Issue And Execution Plans

本目录用于存放**尚未完成、需要持续维护直到关闭**的执行计划。

放在这里的文档遵循这几个规则：

- 只记录仍在推进中的事项，不记录已经完成的当前真相
- 每份计划都应包含范围、分轨任务、优先级、完成标准和迁移条件
- 计划完成后，应从本目录移出，并按内容性质迁入：
  - `docs/current/`：如果它沉淀为当前行为、部署或产品口径
  - `docs/archive/`：如果它只是阶段性执行记录
